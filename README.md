Barberia-Tech 💈
Aplicación Web Corporativa y de Gestión de Turnos desarrollada para Barberías y Peluquerías Masculinas.

🚀 Características del Proyecto:
Interfaz Interactiva: Catálogo visual con servicios de cortes de pelo, tratamientos faciales y atención para niños.
Gestión Dinámica de Citas: Sistema integrado mediante ventanas modales para agendar el día, horario y el profesional elegido de forma simple.
Funcionalidad Flexible: Opción integrada para la cancelación rápida de citas activas.
Atención Automatizada: Redirección directa a canales de soporte y consulta de precios vía WhatsApp.

🛠️ Tecnologías utilizadas:
HTML5, CSS3 y JavaScript puro (Manipulación dinámica del DOM y control de estados de los turnos). Node.js (Entorno de ejecución).

* Proyecto para el curso de Desarrollo Full Stack - Talento Tech (2024).
